로 이어지는 위계 질서가 있다. personal/*.md는 사용자가 직접 작성한 것으로 절대 수정하지 않고 절대적으로 따른다. 어기는 것이 나을 것으로 판단되는 경우 사용자에게 질문을 통해 허락을 받는다.
personal/*.md 를 바탕으로 plans/*.md를 작성, 수정한다.
personal/*.md, plans/*.md를 근거로 하여 실제 코드 파일을 작성한다.

코드 작성 시 변수가 들어가는 경우 터미널 인자로 받지 않도록 한다. 대신 코드 최상단에 변수들을 모아두어 사용자가 쉽게 수정할 수 있도록 한다.
